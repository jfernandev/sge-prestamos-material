# Frontend (Vue.js)

Este directorio contiene el frontend del proyecto desarrollado con Vue 3 y Vite.

## Requisitos

- Node.js 18+
- npm

## Instalación

```bash
npm install
```

## Ejecución en desarrollo

```bash
npm run dev
```

El frontend se sirve por defecto en:

- http://localhost:5173/

## Build de producción

```bash
npm run build
```

## Notas

- El backend (Django) debe estar ejecutándose para que la aplicación cargue datos.
- Por defecto, el frontend consume la API en `http://localhost:8000/api`.
